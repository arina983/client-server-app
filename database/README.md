# База данных проекта

Данный проект использует **PostgreSQL** для хранения данных о качестве сигнала.

### Схема данных:

* **device_data**: Хранит основные данные о местоположении и состоянии устройства в момент замера. Каждая строка это уникальное событие во времени и пространстве.
* **cells_history**: Связанная таблица (**1:N**), хранит подробный список всех видимых сот (PCI) для каждой записи из `device_data`.

### Механизмы целостности данных

В БД внедрены два важных механизма:

1.  **Foreign Key Constraint**: Гарантирует, что данные в `cells_history` не могут существовать без привязки к конкретной точке в `device_data`.
2.  **ON DELETE CASCADE**: При удалении записи о местоположении база автоматически удаляет все связанные с ней записи о PCI.

### Как развернуть локально:

```bash
psql -h localhost -U postgres -d mobile_data -f schema.sql
```
